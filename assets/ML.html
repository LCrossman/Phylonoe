ML biomarkers

ML Biomarker Search:
Online machine learning biomarker identification tool from protein sequences

About

ML Biomarker Search is a program for unix-like operating systems.
It is a computational tool used to train a machine learning model through ensemble methods on decision trees
The latest version is 0.1 (2024/Feb). Downloading from Google Drive to this service uses OAuth 2.0 Secure Authentication Protocol. Https is used to communicate with Google Drive servers which helps ensure security of data transmission during the download process.

Notes on Usage

The number of categories and files are important for success, more categories and longer genomes could mean more files are required for better success in determining results.

Input Format*

1. List of filenames (.fofn) in text format, a list of fofn filenames for each category separated by commas.  You'll need one .fofn for each of your categories to test separated by commas, e.g. for four categories: proc1.fofn,proc2.fofn,proc3.fofn,proc4.fofn 


2. A google drive folder url containing both the full .fofn files, text files one for each category containing a list of filenames of the proteins fasta one per line. Example .fofn file example (example.fofn)
The rest of the input files should be of protein fasta (.faa) one file for each genome sequence for testing, see protein fasta (example.faa)


3. A short list in text format of the categories required separated by commas e.g. 1,2,3,4
